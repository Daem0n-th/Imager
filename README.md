# Imager
A python script which can convert any files to images.
```
usage: imager.py [-h] -f FILE -o OUT [-d]

Encode any file to an image and decode . Supported Image formats are
BMP,JPEG,JPG,PNG,TIFF

required arguments:
  -h, --help            show this help message and exit
  -f FILE, --file FILE  Input binary file or Image file in case of decoding
  -o OUT, --out OUT     Output Image/binary file destination with file extension.
  -d, --decode          Use this to decode the image into a binary file
```
